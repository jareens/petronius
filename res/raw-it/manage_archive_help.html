1 Introduzione

Questa pagina permette di exportare l'archivio. È possibile salvare tutte le definizioni usate da Petronius in un file XML sulla scheda SD. I nome elencati sono i file XML salvati in precedenza. Se si preme brevemente su un nome file, il suo contenuto sarà caricato al posto dei dati correnti: fare attenzione che i dati precedenti saranno cancellati; di conseguenza è consigliabile fare un salvo aggiornato prima di caricare un archivio esportato.
