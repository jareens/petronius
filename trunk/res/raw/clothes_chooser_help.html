1 Introduction

This form is used to insert the values influencing the clothes choice.

2 How to compile the fields

2.1 Date
Insert the date the choice will refer to.
2.2 Elegance
Insert the values for the minumum and maximum elegance level of the clothes you are going to wear. Set the values according to the events you are going to be involved in.
2.3 Season
Set the season of the year you are in.
2.4 Weather
Set the expected weather, or "All" if unsure.
2.5 Garment types
Select the garment types which will be chosen.
