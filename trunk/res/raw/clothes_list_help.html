1 Introduction

This page shows the list of clothes you definded. If you short press a garment its properties will be shown.

2 Options menu

These are the options you can select:

2.1 Choose clothes
Shows a new screen in which you can enter the parameters to let Petronius choose the clothes to wear.
2.2 New garment
Opens a form to enter a new garment's data.

3 Context menu

These are the options listed when you long press a garment:

3.1 Edit garment
Opens a form to edit garment's data.
3.2 Delete garment
Click on it to delete the garment.
3.3 Set available
Sets the garment as available for choices.
3.4 Set unavailable
Sets the garment as unavailable for choices.
3.5 Change image
Allows you to choose another image to represent the garment in the list.
3.6 Reset default image
Resets the garment image to the default one.
